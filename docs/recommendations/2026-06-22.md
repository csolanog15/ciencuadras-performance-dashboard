## Actividades Recomendadas del Dia

### P0 - LCP Mobile en Detalles de Inmuebles (13-15s -> target 2.5s)

**Problema:** Las imagenes del carrusel de fotos del inmueble son el LCP element. Se cargan sin optimizacion, sin dimensiones definidas, y probablemente sin CDN.

**Acciones tecnicas:**
1. Implementar `loading="eager"` en la primera imagen del carrusel y `loading="lazy"` en las demas.
2. Agregar `fetchpriority="high"` al primer `<img>` del componente de galeria.
3. Servir imagenes en formato WebP/AVIF con fallback a JPEG via CloudFront content negotiation.
4. Definir `width` y `height` en todos los `<img>` para evitar CLS.
5. Implementar `<link rel="preload" as="image">` en el `<head>` para la imagen principal.

**Impacto estimado:** LCP -8s (de 14s a ~6s), CLS -0.08.

### P1 - TBT: JavaScript Bloqueante (280-2210ms -> target 200ms)

**Problema:** El bundle principal de Angular se ejecuta sincronicamente, bloqueando el main thread. En el peor caso llega a 2.2s de TBT.

**Acciones tecnicas:**
1. Audit del bundle con `npx webpack-bundle-analyzer`.
2. Lazy load de modulos no criticos: mapa, calculadora de credito, compartir/favoritos.
3. Defer third-party scripts: GTM, Hotjar, chat widgets.
4. Tree shaking agresivo: importar operadores RxJS individuales.
5. Code splitting por ruta.

**Impacto estimado:** TBT -60% (de 800ms a ~320ms).

### P2 - CLS Mobile > 0.1 en Resultados y Detalles

**Problema:** Imagenes sin dimensiones, web fonts sin fallback, elementos dinamicos post-render.

**Acciones tecnicas:**
1. Reservar espacio con `aspect-ratio: 4/3` en imagenes.
2. `font-display: swap` con fallback system fonts.
3. Skeleton screens para cards de resultados.
4. `min-height` para banners dinamicos.

**Impacto estimado:** CLS de 0.107 -> 0.03.

### P3 - LCP Home Mobile (9s -> target 2.5s)

**Problema:** Hero image pesada + SSR no configurado.

**Acciones tecnicas:**
1. Optimizar hero image: WebP, srcset responsive.
2. Preconnect a CDN.
3. Critical CSS inline.
4. Evaluar SSR/Prerendering con Angular Universal.

**Impacto estimado:** LCP -5s (de 9s a ~4s).
